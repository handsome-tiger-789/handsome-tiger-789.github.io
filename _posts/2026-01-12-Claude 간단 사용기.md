좋은 기회로 claude 를 잠깐 사용해볼 기회가 생겼다 <br>
claude cli 는 무료 버전이 존재하지 않아 아직 테스트 해보지 않았다. <br>
<br>

설치는 맥에서 매우 간단 <br>
node 와 npm 이 설치 되어 있어야 하며, 버전이 낮을 경우 claude-code 설치 중 몇 버전으로 설치하라고 친절히 알려줌 <br>
<br>

`npm install -g @anthropic-ai/claude-code` 으로 터미널에 claude 설치 <br>
<br>

이후 `claude` 명령어를 입력하면 최초 설정을 완료 한 뒤 터미널에서 claude를 실행할 수 있다. <br>
<br>

#  Intellij 에서 사용
<br>
클루드 코드를 사용해보고 싶었던 이유 중 가장 큰 이유는 코드를 바로 수정해주는 이 기능 때문이었다.<br>
인텔리제이에서 터미널을 열고 `claude` 실행 후 원하는 명령어를 입력하면 된다.<br><br>

아래 코드는 테스트용으로 명령어를 몇가지 입력한 뒤 순식간에 만들어진 코드다 <br>
ex) JPA를 이용하고 h2 DB를 이용해서 id 를 입력하면 id와 name을 응답해주는 REST API를 만들어줘 <br>
오류코드를 enum으로 관리하고 싶어 등.. <br> <br>

![](/assets/img/2026-01-12-claude/claude-code.png) <br/>

프로젝트 파일을 직접 수정해주고 한번에 여러 로직을 수정해줘서 진짜 순식간에 간단한 API가 만들어졌다.<br>
<br>

# Github 연동

이를 이용해서 Github 에 PR 등록까지 해보려 한다. <br>
<br>

`/install-github-app` 를 입력하면 github 작업을 위한 설정이 시작된다. <br>
이후 아래 링크로 가이드를 준다. <br>
[https://github.com/anthropics/claude-code-action/blob/main/docs/setup.md](https://github.com/anthropics/claude-code-action/blob/main/docs/setup.md){:target="_blank"} <br>

<br>
어떤 사람은 .github/workflows/claude.yml 파일이 자동 생성 된다고 하는데 나는 git cli 가 설치되지 않아서 그랬는지 자동생성 되진 않았다. <br><br>

위 가이드 순서대로 진행하면 된다.
1. github 에서 claude를 설치
2. 프로젝트 설정에서 secrets 등록 (이름은 ANTHROPIC_API_KEY 으로 하면 claude.yml 파일 수정 필요 없음)
3. example 프로젝트에서 claude.yml  파일을 다운받아 해당 프로젝트 아래 .github/workflows/claude.yml 으로 복사
4. 이후 pr 등록 명령

<br>
이때 나는 github-cli 가 설치되지 않았다고 하여 github-cli 설치를 추가로 진행, 로그인 후 다시 명령 했다. <br>
<br>

이후 간단한 비교 PR 등록을 명령 했더니 아래와 같이 보기 쉽게 형식에 맞춰 작성하고 등록까지 완료했다. <br>
![](/assets/img/2026-01-12-claude/claude-git-pr.png) <br/>
