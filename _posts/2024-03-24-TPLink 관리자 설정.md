방화벽 설정 : [https://www.tp-link.com/kr/support/faq/2026/](https://www.tp-link.com/kr/support/faq/2026/){:target="_blank"})

<br/>

> WAN : Wide Area Network의 약자로 멀리 떨어진 지역을 서로 연결하는 네트워크 구축 <br/>
> LAN : Local Area Network의 약자로 한정된 지역에서의 네트워크 구축 <br/>
> 즉, 인터넷에 접속하는 것은 WAN 이라고 생각하면 됨 <br/>

<br/>

1. ServiceType 메뉴에서 시작점(Source) port 대역과 대상지점(Destination) port 대역을 지정한다. <br/>
2. IP 주소 메뉴에서 시작점 IP와 대상지점 IP를 각각 추가한다. <br/>
3. IP Gruop에서 (2)에서 추가한 시작점IP, 대상지점IP를 각각 그룹으로 생성한다. <br/>
4. 방화벽 메뉴에서 (3)-시작점IP그룹, 대상지점IP그룹 / (1)-제어할 port 를 선택하고, 우선순위를 작성하여 추가하면 끝난다. <br> <br>

VLAN 설정 방법 : [https://www.tp-link.com/kr/support/faq/3061/](https://www.tp-link.com/kr/support/faq/3061/){:target="_blank"})
